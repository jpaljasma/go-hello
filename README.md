# Introduction to Go programming language

## Hello World

A simple introduction to Go. Creates a web server in port 8080. Does some header handling.

`go run go-hello.go`

Then open http://localhost:8080 to see the output

## Values

Outputting values of different types, working with Slices, aliasing functions.

`go run values.go`

## Variables

`go run variables.go`

## Constants

`go run constants.go`

## Loops

`go run loops.go`

## If-Else

`go run if-else.go`

There is no [ternary if](https://en.wikipedia.org/wiki/%3F:) `?:` in go

## Switch

`go run switch.go`

## Arrays

Closer look at the Arrays in Golang, + some of the internals like array cap, make, and append.
Calculating prime numbers. Some stats like mean, median.

`go run arrays.go`
